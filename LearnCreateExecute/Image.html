The HTML <img> tag is used to display an image.

The <img> tag contains only attributes such as src and alt, and does not have a closing tag.


The scr Attribute

The src is required because it defines the image's path (URL).


The alt Attribute

The alt attribute is required because it provides an alternate text for the image.

The alt attribute should describe the image.


The Width and Height Attribute

The width and height attributes is used to specify the width and height of the image.
