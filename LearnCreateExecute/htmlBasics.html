HTML Basics




HTML Documents:

All HTML documents must start with a document type declaration: <!DOCTYPEhtml>

This declaration informs the browser about the document's type.

The HTML Document will always start with <html> and ends with </html>.

These are called opening and closing tags.

Your HTML would start to show is in between the opening tag <body> and the closing tag </body>

HTML Open and Close Tags

An opening tag in an HTML document defines the start of an element.

<html> is an opening tag while the closing tag includes a forward slash before the name. </html>, indicating that it is the end of an element.

HTML Page Title

A Page Title is the title of your web seen on the browser's tab.

HTML Headings & Paragraphs

Headings are the texts inside your document. <h1> defines the most important Heading while <h6> defines the least important heading.

And the <p> tag defines a paragraph.

HTML Images

Images are defined with <img>.

The source file (scr), alternative (alt), width and height.

HTML Links

The links are defined with <a>

href if the link's destination.
