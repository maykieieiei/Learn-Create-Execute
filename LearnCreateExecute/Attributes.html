Learn HTML Attributes






All HTML elements can have attributes.

Attributes is a piece pf mark up language used to adjust the behavior or display of an HTML element.

Attributes also provide additional information about the elements.


HTML Attributes





The href attribute

The <a> defines a hyper link and the href attribute specifies the URL.


The src attribute

The <img> tag is used to add images to your HTML document and the src attribute specifies the image to be added.


The width and height attributes

The <img> tag would always contain the width and height attributes as it specifies the size of the image to be displayed. (In pixels)


The lang attribute

In the <html> it is recommended to always include the lang attribute to declare the language of the Web page.
