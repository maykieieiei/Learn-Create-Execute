HTML Formatting Elements

Elements that are designed to display texts that have a special meaning.

  • <b> - Bold text
  • <strong> - Important text
  • <i> - Italic Text
  • <em> - Emphasized Text
  • <mark> - Marked Text
  • <small> - Smaller text
  • <del> - Deleted Text
  • <ins> - Inserted Text
  • <sub> - Subscipt Text
  • <sup> - Superscript Text
