Learn HTML Elements






A HTML Element is between the opening tag and the closing tag, such as:

<tagname> Contents Here <tagname>

Elements




Element Examples are

<h1>Heading Message </h1>

<p>Paragraph Message </p>

Elements




Nested HTML Elements


A Nested HTML Element is where an element can contain another element.

For example:

The elements <html> & <body>, contains other elements inside which are the <h1> & <p>.

Empty HTML Elements

There are some cases that an HTML tag does not require a content or a closing tag.

The tag <br>. defines a line break.
